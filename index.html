📲 Install FamilyFund

Your app is currently running as a local file. To install it on your home screen, you need to host it online (free options below):

Option 1 — GitHub Pages (Free)
Upload familyfund.html to a GitHub repo → enable GitHub Pages → open the HTTPS link on your phone → "Add to Home Screen"
Option 2 — Netlify Drop (Free, Instant)
Go to netlify.com/drop on a computer → drag and drop familyfund.html → share the link with your phone → "Add to Home Screen"
Option 3 — Android "Add to Home Screen" now
Tap the Chrome menu (⋮) → "Add to Home screen" — this creates a shortcut. It won't be a full PWA but works offline once cached.
📲 Install FamilyFund

To install on your device, follow these steps:

1️⃣
Tap the Share button
The ⬆ icon at the bottom of Safari (iOS) or the menu on Android Chrome
2️⃣
Tap "Add to Home Screen"
Scroll down in the share sheet to find this option
3️⃣
Tap "Add"
FamilyFund will appear on your home screen like a native app
